🧠  COMMENT FONCTIONNE L'OSINT ?
L'OSINT (Open Source Intelligence) fonctionne comme une immense base de données distribuée construite à partir de sources publiques et ouvertes. À partir de quelques informations initiales — un nom, une adresse email, un pseudonyme — l'OSINT te permet de tirer des fils pour accéder à une quantité bien plus grande d'informations. C'est un effet boule de neige : chaque donnée trouvée devient un nouveau point d'entrée vers d'autres données.

+++ DONNÉES
01
Point d'entrée
Tu pars d'une seule donnée : un pseudo, un email, un numéro. C'est ta première clé. Elle ouvre plusieurs portes en parallèle sur tous les réseaux et bases de données publics.
02
Croisement de sources
Chaque source trouvée (profil réseau, forum, registre WHOIS, fuite de données) révèle de nouvelles informations qui deviennent à leur tour de nouveaux points d'entrée.
03
Effet boule de neige
Une photo de profil → une ville. Un email → des comptes liés. Un pseudo → des fuites de données. Les pièces s'assemblent pour former une image complète d'une cible.
04
Base de données géante
Internet entier est la base de données. Réseaux sociaux, moteurs spécialisés (Shodan, Censys), archives web, registres publics — tout est là, il faut juste savoir comment l'interroger.
